＜＜全体の流れ＞＞
cpp実行 → data生成 → gnuplotで図を生成 → eps出力

/********************/
/*以下各フォルダの説明*/
/********************/

/********************/
1) cpp
＊ガウス過程回帰のサンプルプログラム
実行にはベクトル/行列演算のライブラリ「Eigen」が必要
・gpr_1.cpp
入力が1次元、出力が1次元の場合のサンプルプログラム
・gpr_2.cpp
入力が2次元、出力が1次元の場合のサンプルプログラム

/********************/
2) data
cppフォルダ内のプログラムを実行した際に生成されるデータ
・gp_1_10.txt
入力1次元：訓練データ10の時のガウス過程回帰の結果
・gp_20.txt
入力2次元：訓練データ20×20の時のガウス過程回帰の結果
・gp_20of2.txt
入力2次元：y=2での断面のガウス過程回帰の結果
・sin_noise_10.txt
入力1次元：訓練データ
・sinx_cosy_noise.txt
入力2次元：訓練データ
・y_2.txt
入力2次元：y=2の断面の描画用

/********************/
3) eps
gnuplotで生成したガウス過程回帰の計算結果
・10_f(x).eps
入力1次元：訓練データ10の時のガウス過程回帰の結果
・20_f(x)_y_2.eps
入力2次元：訓練データ20×20の時のガウス過程回帰のy=2の断面
・20_f(xy).eps
入力2次元：訓練データ20×20の時のガウス過程回帰の結果
・20_f(xy)_cut.eps
入力2次元：訓練データ20×20の時のガウス過程回帰の結果（y=2の切断面も描画）

/********************/
4) gnuplot
計算結果を図示するためにgnuplotを利用します。
・2次元回帰.txt
入力1次元のガウス過程回帰の結果の図を生成(10_f(x).eps)
・3次元回帰.txt
入力2次元のガウス過程回帰の結果の図を生成(20_f(x)_y_2.eps, 20_f(xy).eps
,20_f(xy)_cut.eps)
